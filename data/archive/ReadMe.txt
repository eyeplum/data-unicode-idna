This directory contains an archival record of an erroneously
published conformance test data file for Version 7.0.0 of
UTS #46, Unicode IDNA Compatibility Processing.
 
http://www.unicode.org/reports/tr46/tr46-13.html

IdnaTestOriginal20140506.txt was the originally published version
of IdnaTest.txt. After publication, it was discovered that the
test data had been generated from an earlier version of
IdnaMappingTable.txt, rather than the version of IdnaMappingTable.txt
released for Version 7.0.0 of UTS #46. IdnaTest.txt has
subsequently been corrected to contain the appropriately generated
conformance test examples.

IdnaTestOriginal20140506.txt is preserved here in this archival
directory as a record of the error and for permanent reference
from the erratum notice on this topic. It should not be used
for actual testing of implementations of Version 7.0.0 of
UTS #46.
